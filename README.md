# zaqpolnite

## Порядок установки

```
1. git clone https://github.com/trading-developer/zapolnitut
2. cd zapolnitut && cd docker 
3. make init
4. cd ..
5. yarn install
7. yarn run dev - запуск локальной версии
8. yarn run build
```

## Команды

1. make tests - запуск тестов
